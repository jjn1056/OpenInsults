Standard Portlets

There are two standard portlets for the website, each having several css mixin classes that can alter the appearance. Because mixins can be combined this means that the two basic themes can have potentially dozens of variations. Please the the Zoomwit style guideslines for rules about with themes and color combinations should be used under what situations.

[% WRAPPER $row; WRAPPER $cell type = 'left' classes = 'column_33percent_width'; WRAPPER $gutter type = 'default' sides = 'right'; WRAPPER $portlet title = "Default PortletA" %] Hello this is the default portlet! Feel my power and tremble. [% END; END; END %] [% WRAPPER $cell type = 'left' classes = 'column_33percent_width' %] [% WRAPPER $gutter type = 'default' sides = ['right','left'] %] [% WRAPPER $portlet title = "Default PortletB" %] Hello this is the default portlet! Feel my power and tremble. [% END %] [% END %] [% END %] [% WRAPPER $cell type = 'right' classes = 'column_33percent_width' %] [% WRAPPER $gutter type='default' sides='left' %] [% WRAPPER $portlet type = "simple_portlet" title = "Simple Portlet" %] Hello this is the Simple portlet! Feel my power and tremble. [% END %] [% END %] [% END %] [% END %]

Color Mixins

There is a color mixin class for each of the basic colors of Zoomwit. These color mixins are primarily designed for the default portlet theme. Each color is assigned to to content type, please see the style developer guidelines for more information

[% WRAPPER $row; WRAPPER $column classes="column_33percent_width"; WRAPPER $gutter sides=['right']; WRAPPER $portlet title = 'Gold Portlet' theme = 'gold'; debug_content_short; END; WRAPPER $portlet title = 'Dark Blue Portlet' theme = 'darkblue'; debug_content_short; END; WRAPPER $portlet title = 'Purple Portlet' theme = 'purple'; debug_content_short; END; END; END; WRAPPER $column classes="column_33percent_width"; WRAPPER $gutter sides=['right','left']; WRAPPER $portlet title = 'Green Portlet' theme = 'green'; debug_content_short; END; WRAPPER $portlet title = 'Orange Portlet' theme = 'orange'; debug_content_short; END; WRAPPER $portlet title = 'Grey Portlet' theme = 'grey'; debug_content_short; END; END; END; WRAPPER $column classes="column_33percent_width" type="right"; WRAPPER $gutter sides=['left']; WRAPPER $portlet title = 'White Portlet' theme = 'white'; debug_content_short; END; WRAPPER $portlet title = 'Red Portlet' theme = 'red'; debug_content_short; END; WRAPPER $portlet title = 'Darkerblue Portlet' theme = 'darkerblue'; debug_content_short; END; END; END; END %]

Solid Colors

Each color can be rendered in a solid version. Typically we combine the solid with the rounded corners since this makes the portlet theme more distinctive and different from the standards.

[% WRAPPER $row; WRAPPER $column classes="column_33percent_width"; WRAPPER $gutter sides=['right']; WRAPPER $portlet title = 'Gold Portlet' theme = 'gold' mixins = ['solid','round_corners']; debug_content_short; END; WRAPPER $portlet title = 'Dark Blue Portlet' theme = 'darkblue' mixins = ['solid','round_corners']; debug_content_short; END; WRAPPER $portlet title = 'Purple Portlet' theme = 'purple' mixins = ['solid','round_corners']; debug_content_short; END; END; END; WRAPPER $column classes="column_33percent_width"; WRAPPER $gutter sides=['right','left']; WRAPPER $portlet title = 'Green Portlet' theme = 'green' mixins = ['solid','round_corners']; debug_content_short; END; WRAPPER $portlet title = 'Orange Portlet' theme = 'orange' mixins = ['solid','round_corners']; debug_content_short; END; WRAPPER $portlet title = 'Grey Portlet' theme = 'grey' mixins = ['solid','round_corners']; debug_content_short; END; END; END; WRAPPER $column classes="column_33percent_width" type="right"; WRAPPER $gutter sides=['left']; WRAPPER $portlet title = 'White Portlet' theme = 'white' mixins = ['solid','round_corners']; debug_content_short; END; WRAPPER $portlet title = 'Red Portlet' theme = 'red' mixins = ['solid','round_corners']; debug_content_short; END; WRAPPER $portlet title = 'Darkerblue Portlet' theme = 'darkerblue' mixins = ['solid','round_corners']; debug_content_short; END; END; END; END %]

Style Mixins

Here are the style mixin options.

[% WRAPPER $row; WRAPPER $column classes="column_33percent_width"; WRAPPER $gutter sides=['right']; WRAPPER $portlet title = 'Headless Portlet' theme = 'gold' mixins = 'headless'; %]

This is the headless portlet, for when you don't need a portlet title but want to keep the border.

[% debug_content_short %] [% END; WRAPPER $portlet title = 'Dark Blue Portlet' theme = 'darkblue' mixins = 'no_decoration'; %]

No decoration, which removes the head and the border, but keeps the default internal padding.

[% debug_content_short %] [% END; WRAPPER $portlet title = 'No Internal Padding' theme = 'purple' mixins = 'nopadding'; %]

No internal padding. Good for when you have a table inside the portlet. We also have one and two pixil internal padding which is not demoed here.

[% debug_content_short %] [% END; END; END; WRAPPER $column classes="column_33percent_width"; WRAPPER $gutter sides=['right','left']; WRAPPER $portlet title = 'Upper Case Title' theme = 'green' mixins = 'uppercase_title'; %]

For when you want to force the title to UPPERCASE.

[% debug_content_short %] [% END; WRAPPER $portlet title = 'Lowercase Portlet' theme = 'orange' mixins = 'lowercase_title'; %]

For when you want to force the title to lowercase.

[% debug_content_short %] [% END; WRAPPER $portlet title = 'Titlecase Portlet' theme = 'grey' mixins = 'titlecase_title'; %]

For when you want to force the title to Title Case.

[% debug_content_short %] [% END; END; END; WRAPPER $column classes="column_33percent_width" type="right"; WRAPPER $gutter sides=['left']; WRAPPER $portlet title = 'Centered Title' theme = 'white' mixins = ['title_centered', 'content_centered']; %]

Force the title to center. This also has the 'content_centered' mixin which makes the content inside a portlet centered as well.

[% debug_content_short %] [% END; WRAPPER $portlet title = 'Dotted Border' theme = 'red' mixins = 'dotted_border'; %]

Make the bordering dotted. Good when you want to de-emphasis a portlet

Make the bordering dotted. Good when you want to de-emphasis a portlet
