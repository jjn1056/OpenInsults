Standard Portlet Level Controls

Zoomwit has several sets of dynamic controls that you can easily add to your designs simply by using the correct CSS class (no javascript required). Use these controls as needed.

Accordian Portlets.

You can collapse a stack of portlets into an accordian control. By default all the panes are closed. By default only one pane at a time can be open.

Accordian Portlets Options 1.

You can use the 'selected' class on a portlet to have that portlet open when the page loads. you can have as many selected portlets as you want. However, a selected portlet will close once the user selects a different pane.

Accordian Portlets Options 2.

You can use the 'stuck' class on a portlet to have that portlet open when the page loads. you can have as many stuck portlets as you want. A stuck portlet will never close.

Internal Controls

You can also place controls inside a portlet.

One line

Use the one line control when you want to restrict a block level element, such as a 'p' paragraph, to a single line.

Inner Accordian

Sometimes you want an accordian control inside the portlet itself. This gives you a bit more flexibility in styling and theming. The inner accordian supports the same 'stuck' and 'selected' classes as does the accordian portlet container. Also you can theme the according using the same basic Zoomwit colors, gold, darkblue, etc.
